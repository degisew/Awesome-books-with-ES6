# Awesome-books-with-ES6
A simple website implemented with ES6 basic syntax and features that adds and removes books dynamically.
## Built With the following

- HTML
- CSS
- Modern javaScript(ES6)
- Linters

## Get Started

Please get your browser update.
To get a local copy up and running follow these simple steps.

Run the command below from the Terminal:

      git clone <URL> 'https://github.com/degisew/awesome-books-with-ES6' in your terminal
      
      cd ES6-feature
